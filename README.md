# salcode Neovim Configuration

## Setup

### 1. Make Backup of Existing files in `~/code/nvim`

### 2. Remove Existing `~/code/nvim`

```
rm -rf ~/code/nvim
```

### 3. Clone repo

Clone repo, e.g.

```
git clone git@github.com:salcode/salcode-nvim.git ~/salcode-nvim
```

### Create Symlink

e.g.

```
ln -sf ~/salcode-nvim ~/.config/nvim
```
